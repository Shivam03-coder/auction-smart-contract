# Auction Smart Contract

## Overview

This repository contains a Solidity smart contract for managing an auction. The contract allows participants to place bids, and it handles auction state transitions including starting, running, ending, and canceling the auction.

## Features

- **Start Auction**: Initialize the auction with parameters such as the end block.
- **Place Bid**: Allows participants to place bids during the auction period.
- **End Auction**: End the auction automatically or manually after the specified block.
- **Cancel Auction**: The auction manager can cancel the auction if neede
